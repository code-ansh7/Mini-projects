# 📏 Smart Unit Converter

A console-based unit conversion application built using C++ that allows users to convert Length, Weight, and Temperature values through an interactive menu-driven interface.

## 🚀 Features

* Length Conversion
* Weight Conversion
* Temperature Conversion
* Interactive Menu System
* Loading Screen Animation
* Colored Console Interface
* Multiple Conversions in a Single Run
* User-Friendly Experience

## 🛠️ Tech Stack

* C++
* Functions
* Switch Case
* Loops
* User Input Handling
* Windows Console API (`windows.h`)

## 📚 Conversion Categories

### 📏 Length Conversion

Convert:

* Centimeters → Meters
* Centimeters → Kilometers
* Centimeters → Inches

### ⚖️ Weight Conversion

Convert:

* Kilograms → Grams
* Kilograms → Pounds
* Kilograms → Ounces

### 🌡️ Temperature Conversion

Convert:

* Celsius → Fahrenheit
* Celsius → Kelvin

## 🎯 Project Workflow

1. Launch the application.
2. Loading screen is displayed.
3. Select a conversion category.
4. Enter the required value.
5. View converted results instantly.
6. Choose whether to perform another conversion.
7. Exit the application when finished.

## 📊 Supported Conversions

| Category    | Input Unit      | Output Units             |
| ----------- | --------------- | ------------------------ |
| Length      | Centimeter (cm) | Meter, Kilometer, Inches |
| Weight      | Kilogram (kg)   | Gram, Pounds, Ounces     |
| Temperature | Celsius (°C)    | Fahrenheit, Kelvin       |
