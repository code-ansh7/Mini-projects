# 🏫 School Connect

A modern school management prototype built using **HTML, CSS, JavaScript, and Firebase**.

School Connect is designed to provide students with a centralized platform where they can access important academic information such as notices, homework, notes, timetables, exams, and profile details.

---

## 🚀 Features

### 🔐 Login System

* Student Login Interface
* Name & Mobile Number Input
* Data Storage using Firebase

### 📊 Dashboard

* Personalized Welcome Screen
* Quick Access Navigation
* School Information Overview

### 📢 Notices

* View Important School Announcements
* Centralized Notice Section

### 📝 Homework

* Access Subject-wise Homework
* Organized Academic Tasks

### 📚 Notes

* Study Material Section
* Easy Access to Learning Resources

### 📅 Timetable

* Class Schedule Management
* Daily Academic Planning

### 🎯 Exams

* Exam Information Section
* Upcoming Exam Details

### 👤 Student Profile

* Student Information Display
* Mobile Number & Basic Details
