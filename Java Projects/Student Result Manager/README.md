# 🎓 Student Result Management System

A menu-driven Java console application that manages students' marks efficiently. The project allows users to add marks, display results, calculate statistics, search marks, and analyze pass/fail records using Core Java concepts.

This project was built as part of my **Core Java Learning Journey** to strengthen problem-solving skills through real-world programming.

---

# 🎯 Project Objective

The main objective of this project is to practice Core Java by building a result management system capable of:

* Managing student marks
* Displaying complete results
* Finding highest and lowest marks
* Calculating average marks
* Searching marks
* Counting pass and fail students

---

# 🚀 Features

### ➕ Add Marks

Store marks of multiple students.

---

### 📋 Display Marks

Display marks of all students.

---

### 🏆 Highest Marks

Find the highest marks and display every student who scored them.

---

### 📉 Lowest Marks

Find the lowest marks and display every student who scored them.

---

### 📊 Average Marks

Calculate the average marks of all students.

---

### 🔍 Search Marks

Search a specific mark and display all matching students.

---

### ✅ Pass / ❌ Fail Counter

Count total pass and fail students.

(Current passing marks: **40**)

---

### 🚪 Exit

Safely terminate the application.
