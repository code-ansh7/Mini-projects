# 💰 Hostel Expense Tracker

A menu-driven Java console application that helps hostel students record, manage, and analyze their daily expenses. The project demonstrates the use of arrays, methods, loops, conditional statements, searching algorithms, and modular programming in Core Java.

---

# 🎯 Project Objective

The main objective of this project is to practice Core Java concepts by building a real-world expense tracking application that allows users to:

* Store daily expenses
* View all recorded expenses
* Calculate total and average expenses
* Find the highest and lowest expenses
* Search for a specific expense
* Manage expenses through a menu-driven interface

---

# 🚀 Features

### ➕ Add Daily Expenses

Store daily expenses for any number of days entered by the user.

### 📋 Display Expenses

Display all recorded expenses day by day.

### 💵 Total Expense

Calculate the total amount spent during the selected period.

### 📈 Highest Expense

Find the highest expense and display all days having that highest value.

### 📉 Lowest Expense

Find the lowest expense and display all days having that lowest value.

### 🔍 Search Expense

Search a specific expense amount and display all matching days.

### 📊 Average Expense

Calculate the average daily expense.

### 🚪 Exit

Exit the application safely.
