# 🎒 Student Utility Tool

A menu-driven Java console application that provides multiple daily-use utilities for students. This project combines various Core Java concepts such as loops, conditional statements, switch-case, nested switch, and user input into a single application.

It was built as part of my **Core Java Learning Journey** to practice programming logic and improve problem-solving skills through real-world mini utilities.

---

# 🎯 Project Objective

The objective of this project is to build a multi-functional utility tool that helps students perform common mathematical and logical operations while strengthening Core Java fundamentals.

---

# 🚀 Features

### 🔢 Even / Odd Checker

Checks whether a number is even or odd.

---

### 🔄 Reverse Number

Reverses the digits of any given number.

---

### 🔢 Count Digits

Counts the total number of digits in a number.

---

### ➕ Sum of Digits

Calculates the sum of all digits of a number.

---

### ✖️ Factorial Calculator

Finds the factorial of a given number using loops.

---

### 📋 Multiplication Table

Generates the multiplication table of any number.

---

### 🧮 Calculator

Supports:

* Addition
* Subtraction
* Multiplication
* Division

using a nested menu system.

---

### 🔺 Largest of Three Numbers

Finds the largest number among three user inputs.

---

### 📅 Leap Year Checker

Determines whether a given year is a leap year.

---

### 🎓 Grade Calculator

Assigns grades based on student marks.

Current grading system:

* A
* B
* C
* D
* Fail

---

### 🚪 Exit

Safely exits the application.
