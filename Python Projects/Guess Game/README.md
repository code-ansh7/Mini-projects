# 🎮 Number Guessing Game

A simple and interactive Python console game where the player has to guess a randomly generated number within a given range. The game provides hints after every incorrect guess and tracks the total number of attempts required to win.

This project was developed as part of my programming journey to strengthen logical thinking, user interaction, exception handling, and control flow in Python.

---

# 🎯 Project Objective

The objective of this project is to build a fun console-based game while practicing Python fundamentals such as:

* Random number generation
* User input handling
* Loops
* Conditional statements
* Exception handling
* Game logic implementation

---

# 🚀 Features

### 🎲 Random Number Generation

Generates a random number between **1 and 10** using Python's `random` module.

---

### 🎮 Interactive Gameplay

Allows the player to keep guessing until the correct number is found.

---

### 💡 Smart Hints

Provides hints after every incorrect guess.

* 📈 Higher number please!
* 📉 Lower number please!

---

### 🔢 Attempt Counter

Counts the total number of attempts taken by the player.

---

### 🛡 Input Validation

Handles invalid inputs using `try-except` to prevent program crashes.

---

### 🎉 Winning Message

Displays a congratulatory message along with the total attempts after guessing the correct number.

# 🏗 Project Workflow

```text
Start
   │
   ▼
Generate Random Number
   │
   ▼
Take User Input
   │
   ▼
Compare Guess
   │
   ├── Too High → Show "Lower"
   ├── Too Low → Show "Higher"
   └── Correct → End Game
   │
   ▼
Display Attempts
```
